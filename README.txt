AI Core Engine — Bassam Edition
-------------------------------
نسخة جاهزة كاملة لتشغيل التطبيق مباشرة (محرك ذكاء اصطناعي بسيط مع واجهة عربية).
طريقة التشغيل:
1) pip install -r requirements.txt
2) export OWNER_PIN=bassam1234
3) python run.py
ثم افتح http://localhost:8000

رفع الملفات يتطلب X-Owner-Pin بنفس القيمة.
